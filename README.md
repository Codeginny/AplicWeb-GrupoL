##  Instalación y Uso General
Para ejecutar cualquiera de los proyectos contenidos en este repo:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Codeginny/AplicWeb-GrupoL.git
   ```
2. Entrar a la carpeta del TP deseado (ej: cd tp1).
3. Instalar dependencias:
   ```bash
   npm install
   ```
4. Compilar TypeScript:
   ```bash
   npx tsc
   ```
5. Ejecutar:
   ```bash
   node index.js
   ```
